# Xs &amp; Os of London

Fans' Ultimate Playbook to the City. American football fan guide for London — built for visitors flying in for the Union Jack Classic or NFL London Series.

**Powered by** Philly Sports Trips × Passyunk Avenue.

## Stack

Static HTML / CSS / vanilla JS. No build step. Hosted on Vercel.

## Structure

```
index.html              Single-file site (hero, scoreboard, 12 tours, countdown, intel, footer)
assets/images/          Partner logos (PNG)
vercel.json             Vercel config (clean URLs)
```

## Local preview

Open `index.html` directly in a browser, or:

```bash
npx serve .
```

## Deploy

Connected to Vercel → auto-deploys on push to `main`.
